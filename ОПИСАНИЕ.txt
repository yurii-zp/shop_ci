Приложение представляет собой сокращенный вариант интернет-магазина (не реализован функционал привязки платежной системы).

1) РОЛИ И ПОЛЬЗОВАТЕЛИ.
Все пользователи делятся на две роли: клиеты и администраторы.

2) REGISTRATION & AUTHORIZATION
Реализована с помощью AJAX (используется библиотека jQuery 1.12.4).
Формы 'Sing Up', 'Sign In' и 'Forgot password' подгружаются в одно модальное окно (используется фреймворк Bootstrap 3.3.7).
Валидация на стороне клиента реализована с помощью плагина 'validator.js', заточенного под работу с формами Bootstrap.
В связи с этим CI хэлпер для форм не использовался, т.к. присутствует большое количество элементов оформления формы.
Валидация на стороне сервера реализована с помощью AJAX.

При регистрации есть возможность добавления аватара (хранится в таблице 'users' в формате MEDIUMBLOB. Ограничение размера файла составляет 1MB). Отображается после регистрации в секции header. Если пользователь не добавил аватар, автоматически подставляется placeholder. Есть возможность редактирования аватара в дальнейшем (на странице 'profile', - размещена в dropdown menu секции header после регистрации).

3) ADMIN AREA
Доступна только администраторам
Включает в себя только основной функционал:
- добавление/удаление/поиск категорий товаров (AJAX);
- добавление/удаление/изменение/поиск конкретного товара (AJAX);
для описания товара использован плагин 'ckeditor5 1.0.0 alpha',
есть возможность загрузки изображений (при этом в дирректории assets/images/content создается папка с ID категории товара, которая в свою очередь имеет дочерние папки с ID конкретного товара);
если для конкретного товара изображения не были добавлены, подставляется placeholder. Есть возможность редактирования, которая включает в себя функционал добавления изображений.
- изменение роли/удаление/поиск конкретного пользователя (AJAX);
использованы плагины 'jQuery UI 1.12.1' для создания табов, а также 'Malihu custom scrollbar.js 3.1.5' для добавления полосы прокрутки в таблицы с данными.
Функционал добавления контента для слайдера и описания на Homepage не реализован через админ-панель в целях экономии времени, т.к. не представляет собой основной функционал.

4) HOMEPAGE
Использование плагина 'slick.js 1.6.0' для слайдера.
Основная часть - выводятся последние три товара с каждой категории.

5) CATALOG PAGE
Список всех товаров.
Сверху сортировка по категориям (select) и элемент slider (jQuery UI) для сортировки товаров в пределах выбранной стоимости (AJAX).

6) SINGLE PRODUCT
Представляет собой галерею товара и полное описание (добавляется в админ-панели).
В нижней части страницы блок с отзывами о товаре. Оставить отзыв может только зарегестрированный пользователь. Для оценки товара использован плагин 'raty.js 2.4.5'.

'comments' table:

Headphones - Bose (itemid 17);
Mobile Phones - iPhone X (itemid 12);
Mobile Phones - Samsung Galaxy S8 (itemid 2)
Mobile Phones - Samsung Galaxy S8 plus (itemid 3)

7) CART PAGE
Представляет собой таблицу с добавленными товарами. Возможность удалить/купить товар (AJAX).
Каждый зарегестрированный пользователь имеет стартовую скидку 1%. Процент растет в зависимости от общей суммы покупок. Максимальный процент скидки = 7%.
Стоимость продукта в таблице - без учета скидки.
'Total' - с учетом скидки.

При добавлении товара в корзину, общее количество товара пересчитывается в таблице 'items' (количество задается в админ-панели). Если количество = 0, под товаром появляется надпись 'Not in stock'.
Таким образом, если пользователь добавляет в корзину несколько едениц одного товара, он не добавит больше, чем есть на складе.

При удалении товара из корзины, - пересчитывается 'Total' и общее количество товара Если количество не = 0, надпись 'Not in stock' не показывается.

При покупке товара он удаляется из корзины, данные о пользователе и товаре добавляются в таблицу 'purchases', пользователю пересчитывается % скидки и рейтинг (рейтинг зависит от общей суммы покупок, по умолчанию = 0). Если данного товара больше нет на складе, данные о нем также добавляются в таблицу 'archives'.

Если в корзине нет товаров, появляется сообщение с предложением посетить страницу 'Carts'.

Для незарегестрированного пользователя есть форма с возможностью ввести № заказа и если он добавлял продукты в корзину ранее, они отобразятся, если нет - появляется сообщение с предложением посетить страницу 'Carts'.

8) ARCHIVE PAGE
Доступна только для администраторов.
Представляет собой таблицу с информацией о товарах, которых больше нет на складе. Есть возможность добавить количество товара (AJAX). В этом случае он удаляется из таблицы 'archives' и обновляется в таблице 'items'.

'archives' table:
Headphones - Bose (itemid 17);
Mobile Phones - Samsung Galaxy S8 (itemid 2).

9) PROFILE PAGE
Доступна только для зарегестрированных пользователей.
Представляет собой два блока: 'Edit' и 'Stats'.

Блок 'Edit' - возможность редактировать аватар и email пользователя (AJAX).
Блок 'Stats' - отображает общее количество покупок, общую сумму покупок, текущий процент скидки и текущий рейтинг пользователя.
